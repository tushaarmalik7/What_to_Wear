# 👕 What_To_Wear – Weather-Based Outfit Recommender Chatbot

What_To_Wear is an intelligent chatbot that helps users decide what to wear based on current weather conditions. It provides smart outfit suggestions by analyzing temperature, weather type, and user preferences.

## 🚀 Features
- 🌦️ Real-time weather-based outfit recommendations  
- 🤖 Interactive chatbot interface  
- 📍 Location-based suggestions  
- 👕 Personalized clothing advice (summer, winter, rainy, etc.)  
- ⚡ Fast and simple user experience  

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript  
- Backend: Node.js / Python (update based on your project)  
- API: Weather API (e.g., OpenWeatherMap)  
- Other Tools: Chatbot logic / NLP (if used)  

## 📌 How It Works
1. User enters their location or allows location access  
2. The system fetches real-time weather data  
3. Chatbot analyzes conditions  
4. Outfit suggestions are generated instantly  

## 💡 Example
- Hot Weather → Light cotton clothes, sunglasses  
- Cold Weather → Jackets, hoodies  
- Rainy Weather → Raincoat, waterproof shoes
